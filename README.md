# Shell

## Description

This repository contains shell scripting projects for Holberton School.

## Projects

### io_redirections_and_filters

Scripts about shell I/O redirections, filters and special characters.

| File | Description |
|------|-------------|
| 0-hello_world | Prints Hello, World |
| 1-confused_smiley | Displays a confused smiley |
| 2-hellofile | Displays /etc/passwd |
| 3-twofiles | Displays /etc/passwd and /etc/hosts |
| 4-lastlines | Last 10 lines of /etc/passwd |
| 5-firstlines | First 10 lines of /etc/passwd |
| 6-third_line | Third line of /etc/passwd |
| 7-file | Creates a file with special name |
| 8-cwd_state | Saves ls -la output to a file |
| 9-duplicate_last_line | Duplicates last line of /etc/passwd |
| 10-no_more_js | Deletes all .js files recursively |
| 11-directories | Counts subdirectories |
| 12-newest_files | Shows 10 newest files |
| 13-unique | Prints words that appear once |
| 14-findthatword | Lines containing root in /etc/passwd |
| 15-countthatword | Count occurrences of bin in /etc/passwd |
| 16-whatsnext | Lines with root and 3 following lines |
| 17-hidethisword | Lines without bin in /etc/passwd |
| 18-letteronly | Lines starting with a letter |
| 19-AZ | Replace A with Z and c with e |
| 20-hiago | Remove letters c and C |
| 21-reverse | Reverse input string |
| 22-users_and_homes | Users and home directories sorted |
| 23-empty_casks | Find empty files and directories |
| 24-gifs | List gif files without extension |
| 25-acrostic | Decode acrostic |
| 26-the_biggest_fan | Top 11 hosts from web log |
